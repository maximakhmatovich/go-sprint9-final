# Yandex Practicum Go Goroutines Sprint 9 Final

Этот проект является итогом моего обучения работе с горутинами в Go. Программа демонстрирует параллельную обработку данных для поиска максимального значения в большом массиве случайных чисел.

## Описание

Программа генерирует 100 миллионов случайных целых чисел и находит среди них максимальное значение двумя способами:
- В один поток (последовательно)
- В несколько потоков (используя горутины для параллельной обработки)

## Запуск

Убедитесь, что у вас установлен Go. Затем выполните:

```bash
go run main.go
```

## Результаты

Программа выводит время выполнения для каждого подхода, демонстрируя преимущества параллельной обработки при работе с большими объемами данных.

## Технологии

- Go
- Горутины и синхронизация (sync.WaitGroup)
